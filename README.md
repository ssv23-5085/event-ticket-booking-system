# 🎟 Event Ticket Booking System

A responsive Event Ticket Booking Website built using **HTML, CSS and JavaScript**.

This project allows users to explore events and book tickets through an interactive booking modal with validation.

---

## 🚀 Features

- Responsive design (Mobile, Tablet, Desktop)
- Event cards with images
- Ticket price and venue information
- Remaining tickets indicator
- Booking modal popup
- Auto-select event when clicking "Book Ticket"
- Form validation (name, email, ticket count)
- Ticket availability tracking
- Interactive UI with hover effects

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)
